How to Play
  1. Fill in each cell with a number from 1 to 5
  2. Every row and column contains each number exactly once
  3. The cells in a colored region combine to make the target value using given operation (+, -, x, or รท)
    Note: Numbers is a colored region may occur more than once as long as they don't violate rule #2
Use the number tiles below the puzzle or keyboard to enter numbers into the grid. You may use pencil mode () to mark potential numbers in a cell
Use the Check Puzzle button to verify your solution. If you get stuck, you can reset the puzzle using the Reset Puzzle button.
