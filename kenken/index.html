How to Play
Fill in each cell with a number from 1 to 5.
Every row and every column must contain each number exactly once.
The colored regions have a target number and an operation (+, -, x, or รท).
The numbers in each colored region must combine to match the target number using the operation.
Use the number tiles below the grid or the keyboard to fill in the grid.
You can use pencil mode to mark potential numbers.
